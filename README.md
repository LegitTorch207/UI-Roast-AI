# UI Roast AI

**UI Roast AI** is a full-stack web application that provides **AI-powered feedback on website and app designs**. Users can upload screenshots of their UI, and the system analyzes design elements like **color palettes, typography, spacing, visual hierarchy, and call-to-action effectiveness**, giving actionable suggestions to improve the overall user experience.

---

## Features
- Upload website or app screenshots  
- AI-powered UI/UX analysis  
- Suggestions for colors, fonts, spacing, and alignment  
- Feedback on call-to-action buttons and layout  
- Real-time frontend-backend interaction  

---

## Tech Stack
- **Frontend:** React.js, HTML, CSS, JavaScript  
- **Backend:** Python Flask  
- **AI Integration:** Gemini API / custom AI models  
- **Version Control:** GitHub  

---

## How It Works
1. User uploads a screenshot via the React frontend  
2. Screenshot is sent to the Flask backend  
3. AI analyzes the design for layout, color, typography, and usability  
4. Suggestions are returned and displayed in real-time  

---

## Why This Project
This project helps **designers, developers, and students** quickly identify weaknesses in UI/UX and provides **actionable, AI-driven improvement tips**. It demonstrates **full-stack development and AI integration**.
